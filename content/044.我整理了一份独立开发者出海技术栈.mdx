出海第一步，搞定工具库。

<img src="/assets/044/1.jpg" width="30%" />

最近把一位只会 Vue 的同事掰成 React 程序员了😼。为了让他更快了解 Next.js 和出海需要的技术栈，我根据自己的经验和[@哥飞](https://twitter.com/gefei55) 、[@花果山大圣](https://twitter.com/shengxj1) 分享的信息，整理了一份独立开发者出海工具库发到 Github，24小时突破了500 star。

Github地址：https://github.com/weijunext/indie-hacker-tools

## Web 开发模板

| 模板 | 特性 |
| --- | --- |
| https://github.com/weijunext/smart-excel-ai | Next.js。集成登录、支付（lemon squeezy）、AI功能 |
| https://github.com/wasp-lang/open-saas/ | React + Node.js。集成登录、支付（stripe）、邮件、AI功能 |
| https://shipfa.st/ | Next.js。集成登录、支付（stripe）、邮件、AI功能 |
